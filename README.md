# Web3 Portfolio (Vite + React + Tailwind)

A minimal, dark, Web3-styled personal portfolio page designed for quick deployment.

## Scripts

```bash
# install deps
npm i

# start dev server
npm run dev

# build for production
npm run build

# preview production build
npm run preview
```

## Deploy
- Works on Vercel (auto-detect Vite)
- Also works on Netlify / Cloudflare Pages
